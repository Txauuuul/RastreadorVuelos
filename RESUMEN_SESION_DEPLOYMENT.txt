## 🔑 Variables de Entorno Necesarias

```env
# Amadeus (https://developer.amadeus.com)
AMADEUS_CLIENT_ID=<tu_id>
AMADEUS_CLIENT_SECRET=<tu_secret>

# RapidAPI
RAPIDAPI_KEY=<tu_key>
RAPIDAPI_HOST_FLYSCRAPER2=fly-scraper.p.rapidapi.com
RAPIDAPI_HOST_FLIGHTSSCRAPER=flights-scraper-data.p.rapidapi.com

# Telegram
TELEGRAM_TOKEN=<tu_token_de_@BotFather>

# Database
DATABASE_URL=postgresql://user:password@host/db
# O dejar en blanco para usar SQLite local
```

## 📊 COMPARATIVA: SISTEMA LOCAL vs RENDER

| Característica | Local (Ahora) | Render (Nuevo) |
|---|---|---|
| **Tipo** | Script en tu PC | Servicio en la nube |
| **Disponibilidad** | Solo cuando tu PC está prendido | 24/7/365 |
| **Frecuencia búsqueda** | Cada 5 horas | Cada 5 horas |
| **APIs disponibles** | 3/3 | 3/3 |
| **Costo** | Gratis (energía de tu PC) | Gratis (plan Free) |
| **Escalabilidad** | Limitada (hardware de tu PC) | Ilimitada |
| **DB** | SQLite local | PostgreSQL en la nube |
| **Domain** | Localhost | flight-tracker.onrender.com |
| **Logs** | Terminal local | Dashboard Render |
