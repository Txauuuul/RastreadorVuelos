🚀 PLAN DE ACCIÓN RÁPIDO (COPIAR Y PEGAR)
═════════════════════════════════════════════════════════════════════════════════

1. VERIFICA QUE ESTÁ LISTO:
   python setup_deploy.py

2. CREA GITHUB:
   Ir a https://github.com/signup
   Copiar Personal Access Token

3. SUBE TU CÓDIGO:
   cd c:\XAMP\htdocs\Rastreador
   git init
   git add .
   git commit -m "Flight Tracker v3.0"
   git remote add origin https://github.com/TU_USUARIO/flight-tracker.git
   git branch -M main
   git push -u origin main

4. DEPLOY EN RENDER:
   Ir a https://render.com
   New Web Service → Conecta repositorio
   Agregar variables de .env.example
   Click Create → Espera a "Active"

5. VERIFICA EN TELEGRAM:
   Escribe /start al bot
   Deberías ver el menú

¡LISTO! Ahora está 24/7 en la nube 🎉
