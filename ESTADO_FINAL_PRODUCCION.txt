⏱️  PRÓXIMOS PASOS (ORDEN EXACTO)
════════════════════════════════════════════════════════════════════════════════

PASO 1: GITHUB (5 MINUTOS)
────────────────────────────
1. Ir a https://github.com/signup
2. Crear cuenta con email personal
3. Verificar email
4. Crear nuevo repositorio: "flight-tracker"
5. Copiar tu token en Settings → Developer settings → Tokens

PASO 2: SUBIR CÓDIGO (3 MINUTOS)
────────────────────────────────
PowerShell:

  cd c:\XAMP\htdocs\Rastreador
  git init
  git add .
  git commit -m "Flight Tracker v3.0 - Production Ready"
  git remote add origin https://github.com/TU_USUARIO/flight-tracker.git
  git branch -M main
  git push -u origin main

  (Usuario: TU_USUARIO)
  (Contraseña: PEGA_EL_TOKEN)

PASO 3: RENDER (2 MINUTOS)
──────────────────────────
1. Ir a https://render.com
2. Sign up con GitHub
3. Dashboard → New Web Service
4. Seleccionar repositorio: flight-tracker
5. Nombre: flight-tracker
6. Runtime: Python 3.11
7. Build: pip install -r requirements.txt
8. Start: python run_combined.py
9. Instance: Free
10. Create

PASO 4: VARIABLES DE ENTORNO (1 MINUTO)
───────────────────────────────────────
En Render Dashboard → Environment:
  AMADEUS_CLIENT_ID=<tu_valor>
  AMADEUS_CLIENT_SECRET=<tu_valor>
  RAPIDAPI_KEY=<tu_valor>
  TELEGRAM_TOKEN=<tu_token>

PASO 5: DEPLOY ✅
─────────────────
Click "Save" → Render automáticamente:
  • Descarga código de GitHub
  • Instala dependencias
  • Inicia bot
  • Ejecuta búsquedas cada 5 horas

PASO 6: VERIFICAR (OPCIONAL)
─────────────────────────────
• Enviar mensaje al bot en Telegram
• Ver logs en Render Dashboard
• Esperar a que se ejecute primera búsqueda

Error "Module not found":
  ✓ pip install -r requirements.txt

  ✓ Buscar palabra "error"

  ✓ Verificar TELEGRAM_TOKEN en Render Environment
  ✓ Ver logs: Dashboard → Logs
  ✓ Reintentar: hacer git push nuevamente

  ✓ Verificar AMADEUS/RAPIDAPI keys
  ✓ Verificar suscripciones activas en RapidAPI
  ✓ Copiar keys nuevamente en Render Environment
